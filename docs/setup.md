# Setup

1. Copy `.env.example` to `.env.local`.
2. Set `NEXT_PUBLIC_STORE_MODE=demo` for local development.
3. Leave `CONTACT_EMAIL` empty until a real business email is configured.
4. Install dependencies: `npm install`
5. Run: `npm run dev`
6. Open http://localhost:3000

Optional: configure Stripe test keys, Resend, and Supabase for checkout/email/order persistence.
